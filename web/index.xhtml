¿Que es ADSI?

Analisis y desarrollo de sistemas de información A.D.S.I. ¿Creativo no?
Este maravilloso programa se dedica al diseño de interfaces web, programación orientada a objetos, administración de servidores, administración y programación de bases de datos.
(Sabemos arreglar computadores, pero NO NOS DEDICAMOS A ESO)

Diseño Web
HTML5 y CSS3 Nuestros mejores aliados
La fusión entre el HyperText Markup Language (Estructura general de las paginas web) y las Cascading StyleSheets (El estilo de la pagina, si, las cosas bonitas) le dan forma a cualquier pagina web básica o profesional (De las animaciones y efectos especiales se encarga JavaScript) y son nuestros pinceles a la hora de transformar una simple pagina web, en arte conceptual.
Programación Orienta a Objetos
Java, Php, C# Engranajes del sistema
El diseño web solo es la fachada de la casa, hay que construirla por dentro y de eso se encargan los lenguajes de programación, estos permiten ir mas allá de multiples páginas html bonitas y bien hechas a sistemas complejos que piden, almacenan y muestran tus datos de forma automatica y segura, con solo llenar unos campos y presionar un botón.
Bases de datos
MySQL almacenando nuestros datos desde 1995
¿No te haz preguntado donde quedan los datos que registraste aquella vez en esa página? ... Pues si, van a una base de datos que almacena, inserta, lista y modifica información que quieras guardar en ella. Se puede considerar como la base de un sistema de información (La base y las columnas de una casa) para que este funcione, porque sin datos desaparece el sistema.

La tecnología avanza a pasos gigantes y lo que te mencione es solo es la minima parte de todo el mundo maravilloso que esconde la informatica conoce mas en: Code.org MinTIC
